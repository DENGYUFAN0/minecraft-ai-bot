║        Minecraft AI 机器人  —  安装说明              ║
╚══════════════════════════════════════════════════════╝

📋 第一步：安装两个小程序（只需装一次）
─────────────────────────────────────────────────────
① 安装 Python（编程语言运行环境）
   下载地址：https://www.python.org/downloads/
   → 下载最新版本，安装时勾选 "Add Python to PATH"

② 安装 Node.js（机器人运行环境）
   下载地址：https://nodejs.org/zh-cn
   → 下载 "长期支持版（LTS）"，直接下一步安装即可

─────────────────────────────────────────────────────
🚀 第二步：启动 AI 机器人
─────────────────────────────────────────────────────
• Windows 用户：双击 "启动器.bat"
• Mac 用户：    双击 "启动器.command"
  （如果 Mac 提示"无法打开"，右键 → 打开）

─────────────────────────────────────────────────────
⚙️  第三步：在图形界面里填写配置
─────────────────────────────────────────────────────
服务器地址  →  填你的 Minecraft 服务器 IP
端口        →  通常是 25565，不用改
MC 版本     →  和你的服务器版本对应（如 1.20.1）
机器人名称  →  机器人在游戏里显示的名字
AI 供应商   →  选择你有 API Key 的服务：
               Claude (Anthropic) / GPT (OpenAI)
API Key    →  填入你的 API Key（见下方说明）
机器人目标  →  用中文描述你希望机器人做什么

─────────────────────────────────────────────────────
🔑 如何获取 API Key
─────────────────────────────────────────────────────
Claude API Key（推荐）：
   https://console.anthropic.com/
   注册账号 → API Keys → Create Key
   Key 格式：sk-ant-api03-...

OpenAI API Key：
   https://platform.openai.com/api-keys
   注册账号 → API Keys → Create new secret key
   Key 格式：sk-proj-...

─────────────────────────────────────────────────────
💡 机器人目标示例
─────────────────────────────────────────────────────
• "收集木头和石头，建造一个简单的小木屋"
• "探索地下洞穴，挖掘矿石，越多越好"
• "在游戏里生存尽可能长的时间"
• "跟随玩家 Steve，帮他收集材料"

─────────────────────────────────────────────────────
⚠️  注意事项
─────────────────────────────────────────────────────
• Minecraft 服务器需要开启离线模式（online-mode=false）
  在 server.properties 文件里设置
• 首次启动会自动安装 Bot 组件，需要等待约 1-2 分钟
• 使用 AI API 会产生费用，请注意你的 API 使用量

─────────────────────────────────────────────────────
❓ 在游戏里控制机器人
─────────────────────────────────────────────────────
在聊天框输入：
  !goal 去挖矿      → 改变机器人目标
  !goal 跟着我      → 让机器人跟随你
  !goal 停下来      → 让机器人暂停
